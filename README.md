# HAC-feedback-project-2026

**Srcipts of 'Stress-induced transgenerational plant defense by virus-mediated reprogramming of secondary metabolism in rhizosphere bacteria' in Hu lab**    

Data Analysis Scripts for Microbial Diversity and Correlation

**Data**    
This repository contains data for generating alpha and beta diversity analyses of viral and bacterial communities, as well as data required for correlation calculations.    

**R Scripts**    
votu.codes.R: Script for calculating viral diversity    
bac.codes.R: Script for calculating bacterial diversity    
corrplot.R: Script for visualizing linear correlations    
corr.heatmap.R: Script for generating correlation heatmaps    

**Figures**    
Figures generated from the data using the provided scripts

**Pipeline**  
viral_detection_workflow.sh: Pipeline for generating soil viral communities and identifying AMGs  
MAG_workflow.sh: Pipeline for generating dereplicated MAGs  
host_prediction.sh: Pipeline for virus host prediction
